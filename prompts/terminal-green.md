## [STYLE SPEC]
Use GitHub-dark black, terminal green, and monospace typography as a focused developer interface.
## [LAYOUT RULES]
Structure slides like concise command sessions with clear prompts, syntax emphasis, scan-line restraint, and one blinking-cursor motif.
## [FORBIDDEN]
No fake hacking clichés, long code dumps, bright secondary colors, low-resolution CRT noise, or decorative command output.
